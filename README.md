# STONE Portfolio

Personal portfolio website for overseas content growth, content strategy, and creative operations.

## Tech Stack

- React
- Vite
- CSS animation

## Local Preview

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Deploy

You can deploy this project to GitHub Pages, Vercel, Netlify, or any static hosting platform.

For Vercel / Netlify:

- Build command: `pnpm run build`
- Output directory: `dist`

For Vercel, this project also includes `vercel.json`, so Vercel should detect the correct build output automatically.

If your GitHub repository contains the `STONE-portfolio` folder as a subfolder instead of placing these files at the repository root, set Vercel's Root Directory to:

```txt
STONE-portfolio
```
